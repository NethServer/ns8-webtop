This page provides automatic configuration for supported email clients.
If you reached this page manually, no further action is required.

Autoconfiguration Endpoints

  • Thunderbird
    Uses: autoconfig.{{ domain }}/mail/config-v1.1.xml
    This is the configuration file used by Thunderbird and compatible clients for automatic setup.
  • Outlook
    Uses: autodiscover.{{ domain }}/autodiscover/autodiscover.xml
    This is the configuration endpoint used by Microsoft Outlook for automatic account discovery.
